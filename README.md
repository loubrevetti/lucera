Lucera Application
===================================================================

## Install Application

#### Install and Start Server

1. cd off of root project to server
2. `cd server`
3. Install dependencies 
4. `npm install`
5. Run Node server this command is for Windows machine 
6. `npm run start:server`



#### Install React and UI

1. cd off of root project to server
2. `cd app`
3. Install dependencies 
4. `npm install`
5. Run Node server this command is for Windows machine 
6. `npm run start:app`
